heashotsurt plugin for Big Brother Bot (www.bigbrotherbot.com)
==============================================================

By #UbU#Courgette, inpired from the spree plugin by Walker.


Description
-----------

When a player make a headshot (ie: a kill with last bullet in 
head or helmet), display in console how many headshot the killer 
made so far.


Installation
------------

 * copy headshotsurt.py into b3/extplugins
 * copy headshots.xml into b3/extplugins/conf
 * update your main b3 config file with :

<plugin name="headshotsurt" priority="16" config="@b3/extplugins/conf/headshots.xml"/>
